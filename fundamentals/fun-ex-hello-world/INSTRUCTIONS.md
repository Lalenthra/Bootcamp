# Hello, world!

To get familiar with creating repositories, pushing code, and getting feedback on our work, let's write a simple Hello World program.

Create a Python program that prints "Hello, world!".

To do this:

1) Create a repository in GitLab.
2) Create a file called `project.yaml` with the following content:
```yaml
project: hello-world
```
3) Create a file called `greet.py` that prints exactly `Hello, world!`, followed by your operating system's new line sequence, and then terminates.

We'll be testing this script on Linux, which means we'll be expecting `\n` as the new line sequence.

Remember, the purpose of this project is to learn the basics of how we do projects at WeThinkCode_, including using Git correctly. Please refer to [this material](#) for elaboration on what that means.

Good luck!
